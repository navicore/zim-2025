# masters

Creation
--------

This folder contains finalized, polished versions files and metadata.

The music was all made in Vashon WA, USA between July and October 2025.

The instruments are a mix of Serge 4U and 3U modular synthesizers, Eurorack
modular synthesizers, and field recordings.

The album was mastered by Rupert Clervaux in November 2025.
