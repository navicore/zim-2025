# Album Release Checklist

## 12+ Weeks Before Release

* [x] Document the creation journey (photos, notes, video)
* [x] Decide on release date (Fridays are common in the industry)
* [x] Register songs with a PRO (ASCAP or BMI)
* [x] Register with SoundExchange for mechanical royalties
* [x] Finalize and approve masters

## 8+ Weeks Before Release

* [x] Write liner notes
  * Instrumentation
  * Concept/approach for each track
  * Inspiration behind the album
* [x] Prepare track/album credits
  * Track titles and lengths
  * Composer(s) for each track
  * Performer(s) and instrumentation
  * Recording location and dates
  * Recording, mixing, and mastering engineer(s)
  * Producer(s), collaborators, photographer, etc.
  * Special thanks (optional)
* [x] Create/finalize album artwork (3000x3000 pixels, square)
* [x] Upload to Bandcamp and prepare page
* [x] Submit tracks to digital distributor (DistroKid, etc.)

## 4+ Weeks Before Release

* [ ] Plan release day event (listening party, live stream, show)
* [ ] Announce the release day event
* [ ] Prep social media and email blasts
* [ ] Prep banners and promotional assets
* [ ] Start posting more regularly to build momentum

## 2-3 Weeks Before Release

* [ ] Tease release on social media
* [ ] Share behind-the-scenes footage
* [ ] Post countdowns to build energy
* [ ] Encourage follows on Bandcamp and mailing list subscriptions

## 1 Week Before Release

* [ ] Send personal messages to friends, super fans, and collaborators
* [ ] Prep daily social posts for the week
  * Individual track highlights
  * How it was made
  * Inspiration and collaborators
  * Music snippets
  * Behind-the-scenes content

## Release Day

* [ ] Announce on all social platforms
* [ ] Push "Release" on Bandcamp
* [ ] Send announcement to Bandcamp subscribers
* [ ] Contact Phillip and Modular Station
